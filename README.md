## 🚀 نظرة عامة

موقع شخصي احترافي مشابه لتصميم [younes.expert](https://www.younes.expert/) مع:
- ثيم داكن أنيق (Dark Theme) بلون أحمر مميز
- تصميم متجاوب بالكامل (Responsive)
- دعم اللغتين العربية والإنجليزية
- تحريكات سلسة وتأثيرات بصرية احترافية

## ✅ الأقسام المُنجزة

| القسم | الوصف |
|-------|-------|
| **Navbar** | شريط تنقل ثابت مع تبديل الثيم واللغة |
| **Hero** | صفحة رئيسية بصورة دائرية وإحصائيات |
| **Experience** | الخبرة المهنية والتعليم بتصميم Timeline |
| **Certifications** | شبكة شهادات 3×2 مع شعارات المؤسسات |
| **Projects** | مشاريع البيانات والذكاء الاصطناعي |
| **Tech Specs** | مجالات الخبرة التقنية |
| **Tech Stack** | التقنيات والأدوات المستخدمة |
| **Companies** | جهات العمل (مؤشرات) |
| **Contact** | نموذج تواصل + روابط اجتماعية |
| **Footer** | تذييل بحقوق الملكية والروابط |
